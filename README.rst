Python package to remove spike in 2D images.

Install
-------
With ``pip``:

.. code:: bash

    $ pip install despike

With the ``source files``:

.. code:: bash

    $ git clone https://github.com/seignovert/despike.git
    $ cd despike ; python setup.py install
